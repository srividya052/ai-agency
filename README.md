# AI Nexus – Modern AI Agency Website

A modern, responsive AI agency landing page built using HTML, Tailwind CSS, and JavaScript.

## Live Demo

https://srividya052.github.io/ai-agency/

## Features

* Responsive design for desktop and mobile
* Modern AI-themed UI
* Animated hero section
* Services showcase
* Testimonials carousel
* Blog section
* FAQ accordion
* Contact form
* Professional footer
* GitHub Pages deployment

## Technologies Used

* HTML5
* Tailwind CSS
* JavaScript
* Git
* GitHub Pages
